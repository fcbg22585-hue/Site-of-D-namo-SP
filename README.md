# ⚽ Dínamo SP

Site oficial do clube fictício **Dínamo SP**, criado para representar um projeto de futebol amador com estilo moderno e responsivo.

---

## 🌐 Sobre o projeto

O Dínamo SP é um site de página única (SPA simples) com:

- Página inicial com destaque do clube
- Informações sobre o time
- Elenco de jogadores
- Próximos jogos
- Contato
